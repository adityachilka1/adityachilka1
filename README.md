#### What I'm working on

I make tools for engineers building with LLMs and the Model Context Protocol — the layer Anthropic, OpenAI, and the open-source agent community are quickly standardising around. My pinned repos are where most of my time goes.

- **[`mcp-devtools`](https://github.com/adityachilka1/mcp-devtools)** — Chrome DevTools for the Model Context Protocol. Inspect, profile, replay, and diff every MCP call your agent makes. *(Flagship.)*
- **[`skillforge`](https://github.com/adityachilka1/skillforge)** — CLI + open registry for Claude Skills. Author, version, eval, publish. *(In development.)*
- **[`agentbench`](https://github.com/adityachilka1/agentbench)** — Snapshot tests for AI agent traces. Framework-agnostic — works with CrewAI, LangGraph, Mastra, OpenAI Agents SDK. *(In development.)*

#### A few principles

1. **Local-first beats cloud-first** until proven otherwise. Most dev tools don't need to phone home.
2. **Drop-in beats opinionated.** A user who has to refactor to try your library will not try your library.
3. **Ship the smallest thing that's useful**, then iterate weekly in public.
4. **Read the protocol spec, not the framework docs.** Frameworks come and go. Protocols outlast them.
